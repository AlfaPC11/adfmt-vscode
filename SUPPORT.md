# Support

Use the GitHub issue tracker for reproducible bugs and feature requests:

<https://github.com/AlfaPC11/adfmt-vscode/issues>

Include:

- VS Code version and operating system
- Extension version
- `adfmt --version` output
- Relevant `.adfmt` configuration
- Output from the `adfmt` output channel

This project is maintained by Alfa as an individual, without an organization
or support team. Responses are best-effort and have no guaranteed schedule.
